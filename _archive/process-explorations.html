Process — Layout Explorations

Seven directions for the four-step process (Book / Plan / Capture / Deliver). All Archivo + B/W. Pick a winner and we'll wire it into app/page.tsx.

P1 — Horizontal stepper P2 — Sticky narrative P3 — Numeral panels P4 — Vertical timeline P5 — 2×2 image cards P6 — Typographic index P7 — Scroll-progress band
P1 — Horizontal stepper

Four columns, connecting rule, no images

Clean and instantly scannable. A horizontal rule connects the four steps; each gets a number, name, one-line description. Best when you want the process to feel light and quick.

P2 — Sticky narrative

List on the left, sticky photo pane on the right

Same DNA as the services section — copy scrolls past while a sticky image pane swaps per step. Page reads as one design system top-to-bottom. Hover any row to swap the image.

Massive faded numerals as backdrops

Each step row has a giant Archivo Black numeral ghosted at 6% opacity behind the copy + small image. Numbers carry the design. Strong, branded, very studio-monograph.

P6 — Typographic index

No images, pure type

Number column, big name, description. No imagery — purest typographic treatment, fastest to read. Best when the photos are doing the heavy lifting in the services section above.
